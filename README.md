# HTB Machines Finder

Welcome to the HTB Machines Finder! This tool is inspired by the classes of S4vitar at Hack4U. It is designed to help you search and download machines listed on htbmachines.github.io.

## Features

- **Automated Search:** Quickly search for HTB machines by name or keyword.
- **User-Friendly:** Simple command-line interface for ease of use.

## Requirements

- Bash
- `curl` (for downloading files)
- `jq` (for parsing JSON data)

## Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/yourusername/htb-machines-finder.git
    cd htbmachines
    ```
2. Make the script executable:
    ```bash
    chmod +x htbmachines.sh
    ```

## Usage

Run the script with the desired machine name or keyword:
```bash
./htbmachines.sh -m "machine_name"
